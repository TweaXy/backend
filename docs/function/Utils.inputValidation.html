    static IsEmail(identifier) → {boolean}
    utils/inputValidation.js, line 7

    Checks if the provided identifier is a valid email address.
    Parameters:
    Name Type Description
    identifier string The user identifier to check.
    Returns:
    boolean - - Returns true if the identifier is a valid email address (comprising a sequence of letters, followed by '@', then another sequence of letters, '.', and at least two more letters), otherwise returns false.
    Example
    const isEmail = IsEmail('user@example.com');
